Introduction

Welcome to the Bifold Admin Dashboard documentation. This guide will help you understand how to use the admin dashboard to manage content across the entire website, including the "How It Works" page with its mobile app visualization sequence.

This admin dashboard allows you to edit all text content, upload and replace images, and manage the sequence of steps shown in the "How It Works" page.

Getting Started

Logging In

To access the admin dashboard, navigate to /admin in your browser. You will be prompted to enter your username and password.

The default login credentials are:

  • Username: admin
  • Password: bifold2024

For security reasons, we recommend changing these default credentials after your first login.

Dashboard Overview

After logging in, you will be directed to the main dashboard. Here you can see an overview of your website content, including:

  • Total number of pages
  • Number of app screens in the "How It Works" sequence
  • Total number of images
  • Quick access links to common tasks
  • Recent content updates

The sidebar on the left provides navigation to all sections of the admin dashboard.

Content Management

Editing Pages

To edit content on any page:

  1. Click on the corresponding page link in the sidebar (Home, About, Products, Contact)
  2. Edit the text in the provided form fields
  3. Click "Save Changes" to apply your edits
  4. Use the "Preview" button to see how your changes will look on the live site

All text fields support basic formatting. You can use the rich text editor to add formatting such as bold, italic, lists, and links.

Uploading Images

To replace an existing image:

  1. Navigate to the page containing the image you want to replace
  2. Click on the image to open the image selector
  3. Click "Upload New Image" or select from existing images
  4. Crop and adjust the image as needed
  5. Click "Save" to apply the changes

For best results, use high-quality images with transparent backgrounds (PNG format) whenever possible.

How It Works Page

The "How It Works" page features a sequence of mobile app screens that demonstrate the user journey. This section explains how to manage this sequence.

Editing Steps

To edit an existing step in the sequence:

  1. Navigate to "How It Works" in the sidebar
  2. Find the step you want to edit
  3. Update the title, description, and alt text as needed
  4. To change the image, hover over the current image and click "Change Image"
  5. Click "Save All Changes" when you're done

Reordering Steps

You can change the order of steps in the sequence using the up and down arrow buttons:

  1. Click the up arrow to move a step earlier in the sequence
  2. Click the down arrow to move a step later in the sequence
  3. The step numbers will update automatically
  4. Click "Save All Changes" to apply the new order

Adding/Removing Steps

To add a new step:

  1. Scroll to the bottom of the steps list
  2. Click "Add New Step"
  3. Fill in the title, description, and alt text
  4. Upload an image for the new step
  5. Click "Save All Changes"

To remove a step:

  1. Find the step you want to remove
  2. Click the trash icon in the top-right corner of the step card
  3. Confirm the deletion when prompted
  4. Click "Save All Changes"
Note: You must have at least one step in the sequence. The delete button will be disabled if there is only one step remaining.

Troubleshooting

If you encounter any issues while using the admin dashboard, try these common solutions:

Changes Not Saving

  • Make sure you click "Save Changes" or "Save All Changes" after making edits
  • Check that you have a stable internet connection
  • Try refreshing the page and making the changes again

Image Upload Issues

  • Ensure the image is in a supported format (JPG, PNG, GIF)
  • Check that the image file size is under 5MB
  • Try using a different browser or clearing your browser cache

Login Problems

  • Verify that you're using the correct username and password
  • Check if Caps Lock is enabled
  • Clear your browser cookies and try again

If you continue to experience issues, please contact technical support for assistance.
